# plot wind lease cumulative areas

plot wind_dev_speed. This is shelf wide

## Usage

``` r
plot_wind_dev_speed(shadedRegion = NULL, report = "MidAtlantic")
```

## Arguments

- shadedRegion:

  Numeric vector. Years denoting the shaded region of the plot (most
  recent 10)

- report:

  Character string. Which SOE report ("MidAtlantic", "NewEngland")

## Value

ggplot object
