# Wind energy development speed

Data include the planned wind development areas on the Northeast US
shelf.

## Usage

``` r
wind_dev_speed
```

## Format

Data set contains 94 rows and 5 columns

- Time: Year.

- Var: Cumulative wind lease area.

- Value: Value of variable `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

- Report_year: Year values reported.

## Details

More information including processing and indicator derivation steps are
available at
