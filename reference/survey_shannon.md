# Shannon Diversity of NEFSC Bottom Trawl Survey

Data include shannon diversity values for bottom trawl survey catch.

## Usage

``` r
survey_shannon
```

## Format

436 rows and 5 columns.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

- Time: Year.

- Var: Variable describing shannon diversity.

- Value: Value of variable `Var`.

- Units: Units of variable `Var`.
