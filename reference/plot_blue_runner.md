# plot blue_runner

Creates map of blue runner locations by decade

## Usage

``` r
plot_blue_runner(shadedRegion = NULL, report = "MidAtlantic")
```

## Arguments

- shadedRegion:

  Numeric vector. Years denoting the shaded region of the plot (most
  recent 10)

- report:

  Character string. Which SOE report ("MidAtlantic", "NewEngland")

## Value

ggplot object
