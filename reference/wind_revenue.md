# Fisheries revenue from wind lease areas

Data include Top 5 Species Revenue from Wind Development Areas

## Usage

``` r
wind_revenue
```

## Format

Data set contains 1026 rows and 4 columns

- Time: Year.

- Var: Top 5 Species Revenue from Wind Development Areas.

- Value: Value of variable `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred.
