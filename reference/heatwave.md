# Marine Heatwaves

Data include cumulative intensity, maximum intensity and duration of
surface and bottom water marine heatwaves on the NES.

## Usage

``` r
heatwave
```

## Format

648 rows and 5 columns.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

- Time: Year.

- Var: Cumulative or maximum marine heatwave intensity.

- Value: Value of variable `Var`.

- Units: Units of variable `Var`.

## Details

More information including processing and indicator derivation steps are
available at
<https://noaa-edab.github.io/tech-doc/marine-heatwave.html>.
