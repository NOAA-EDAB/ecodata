# Gray Seal Bycatch

Data include estimated Gray Seal bycatch on the Northeast Shelf.

## Usage

``` r
grayseal
```

## Format

A time series with 130 rows and 4 columns.

- Time: Year

- Var: Bycath variable

- Value: Value of variable

- EPU: Shelfwide indicator

## Details

Methods used to calculate data are available at
<https://noaa-edab.github.io/tech-doc/harbor-porpoise-and-gray-seal-bycatch.html>.
