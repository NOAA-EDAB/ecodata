# Ecological Production Units

Ecological Production Units (EPUs) delineate regions unique in
physiographic, oceanographic and biotic aspects on the Northeast U.S.
Continental Shelf, representing an area of approximately 264,000 km
within the 200 m isobath. There are four EPUs on the Shelf, including
the Mid-Atlantic Bight (MAB), Georges Bank (GB), Gulf of Maine (GOM),
and Scotian Shelf (SS). The Scotian Shelf lies outside the management
domain of US Fishery Management Councils and is generally not considered
in State of the Ecosystem Reports.

## Usage

``` r
epu_sf
```

## Format

This data set is a `sf` object with 4 features and 3 fields.

## Details

More information about the delineation of Ecological Production Units
can be found at <https://noaa-edab.github.io/tech-doc/epu.html>.
