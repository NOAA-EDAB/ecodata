# Species distribution on the NES.

Data include timeseries of depth, distance from shelf and distance along
shelf.

## Usage

``` r
species_dist
```

## Format

Data set contains 275 rows and 5 columns

- Time: Year.

- Var: Depth or Distance along shelf or Distance from shelf.

- Value: Value of variable `Var`.

- Units: Unit of Variable `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

## Details

More information including processing and indicator derivation steps are
available at
<https://noaa-edab.github.io/tech-doc/species-distribution-indicators.html>.
