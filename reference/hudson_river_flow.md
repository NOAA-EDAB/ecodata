# Hudson River Flow

Flow from Hudson River into the Atlantic.

## Usage

``` r
hudson_river_flow
```

## Format

40 rows and 4 columns

- Time: Year.

- Var: Flowrate of Hudson River

- Value: Value of variable `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

## Details

More information about this data set may be found at
<https://noaa-edab.github.io/tech-doc/hudson-river-flow.html>.
