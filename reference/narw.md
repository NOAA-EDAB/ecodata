# North Atlantic right whale abundance

Data include abundance estimates for the North Atlantic right whale
(NARW) provided by the Protected Species Branch of the NEFSC. Abundance
was estimated following the methodology of Pace et. al 2017.

## Usage

``` r
narw
```

## Format

These data include 129 rows and 5 columns.

- Var: Specifies variable of interest, including NARW and calf abundance
  estimate median and 95% credible intervals.

- Value: Value of variable `Var`.

- Units: Units of variable `Var`.

- Time: Year.

- EPU: Ecological Production Unit (EPU).

## Details

More information about these data may be found
<https://noaa-github.io/tech-doc/right-whale-abundance.html>.

## References

Pace, R. M., Corkeron, P. J., Krauss, S. D. (2017) State-space
mark-recapture estimates reveal a recent decline in abundance of North
Atlantic right whales. *Ecology and Evolution*, 1-17.
