# Chesapeake Bay Gridded Sea Surface Temperature

Chesapeake Bay Gridded Sea Surface Temperature

## Usage

``` r
ches_bay_sst
```

## Format

A data set containing 119577 rows and 4 columns.

- Latitude: Latitude of sea surface grid.

- Longitude: Longitude of sea surface grid.

- Value: Tempurature in Celsius.

- Season: Annual Season.
