# Fish Condition

Mean relative condition calculated from the NEFSC bottom trawl survey.

## Usage

``` r
condition
```

## Format

These data include 1612 rows and 4 columns.

- Var: Species ID code

- Value: Value of variable `Var`.

- Units: Units of variable `Var`.

- Time: Year.

## Details

These data are fully documented at
<https://noaa-edab.github.io/tech-doc/fish-condition-indicator.html>.
