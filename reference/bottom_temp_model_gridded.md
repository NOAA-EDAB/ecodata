# Bottom temperature - Seasonal Gridded Product

Spatially modeled bottom temperatures by season on the Northeast
Continental Shelf from most recent year.

## Usage

``` r
bottom_temp_model_gridded
```

## Format

Data contain raster files for each season.

- Raster stack of 4 layers containing bottom temperature anomalies.

## Details

Links to source data, further data definitions, and extended methodology
are available at
<https://noaa-edab.github.io/tech-doc/bottom-temperature---high-resolution.html>
