# SAFMC managed Blue Runner presence on NES

Data include spatial, presence/absence data of South Atlantic FMC
managed species blue runner (Caranx crysos) found on the NES.

## Usage

``` r
blue_runner
```

## Format

Data set includes 1152 rows and columns.

- Var: Either lat or long of Blue runner present on NES broken into 3
  time catgories (prior to 2001, 2001-2010, 2010-pres)

- Value: Value of variable `Var`.

- Time: unused

- Units: decimal degrees

- EPU: Ecological Production Unit (EPU) where sampling occurred.
