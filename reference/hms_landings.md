# Highly Migratory Species Landings and Revenue

Data include landing weight and revenue from group HMS species.

## Usage

``` r
hms_landings
```

## Format

156 rows and 4 columns.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

- Time: Year.

- Var: Species weight or revenue landings.

- Value: Value of variable `Var`.

More information including processing and indicator derivation steps are
