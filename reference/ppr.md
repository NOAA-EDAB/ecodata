# Ecosystem Overfishing Indicators

Data include ecosystem overfishing indicators (Primary Production
Required, Ryther, Fogarty).

## Usage

``` r
ppr
```

## Format

550 rows and 5 columns.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

- Time: Year.

- Var: Ecosystem Overfishing Indicator.

- Value: Value of variable `Var`.

- Units: Units of variable `Var`.

## Details

More information including processing and indicator derivation steps are
available at
<https://noaa-edab.github.io/tech-doc/ecosystem-overfishing.html>.
