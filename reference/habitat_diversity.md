# Habitat Diversity

Habitat diversity from the Northeast Regional Habitat Assessment

## Usage

``` r
habitat_diversity
```

## Format

Time series with 400 rows and 4 columns.

- Time: Year

- Var: Diversity metrics

- Value: Value of Variable

- EPU: Representative Ecological Production Unit (MAB = Mid-Atlantic
  Bight, GB = Georges Bank, GOM = Gulf of Maine, SS = Scotian Shelf, or
  All).

## Details

Methods used to calculate data are available at
<https://noaa-edab.github.io/tech-doc>.
