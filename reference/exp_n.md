# Expected Number of Species per 1000 Individuals

Data include mean and standard deviation of expected number of species
caught on the NEFSC bottom trawl survey per 1000 individuals collected
for both Fall and Spring.

## Usage

``` r
exp_n
```

## Format

909 rows and 6 columns.

- Value: Value of variable `Var`.

- Units: Units of variable `Var`.

- Time: Year.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

- Var: Vessel and Survey season.

## Details

Methods used to calculate data are available at
<https://noaa-edab.github.io/tech-doc/expected-number-of-species.html>.
