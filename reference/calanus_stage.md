# Calanus Stage

Time series of calanus abundance by life stage.

## Usage

``` r
calanus_stage
```

## Format

A data set containing 1006 rows and 5 columns.

- Year: Time

- EPU: Ecological Production Unit (EPU) where data originated. Here
  "MAB" refers to the Mid-Atlantic Bight.

- Value: Value of variable.

- Var: calanus stage

- Units: Number of calanus/meter cubed.
