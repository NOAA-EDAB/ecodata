# Virginia waterbird productivity

These data are time series of productivity for groups of Virginia
waterbirds

## Usage

``` r
seabird_mab
```

## Format

Data set contains 24 rows and 5 columns

- Group: Group of waterbirds.

- Value: Value of variable `Var`.

- Units: Units of variable `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred. All
  data were sampled in southern Gulf of Maine.

- Time: Year of sampling.

More information including processing and indicator derivation steps are
available at
<https://noaa-edab.github.io/tech-doc/ma-waterbird-productivity.html>.
