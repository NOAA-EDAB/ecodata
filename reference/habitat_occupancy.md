# Habitat Occupancy

Data include habitat area with a probability of occupancy greater than
0.5 was modeled for many species throughout the Northeast Large Marine
Ecosystem (NE-LME) using random forest decision tree models.

## Usage

``` r
habitat_occupancy
```

## Format

86 rows and 5 columns.

- Value: Value of variable `Var`.

- Var: Season.

- Time: Time step of `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

## Details

Methods used to calculate data are available at
<https://noaa-edab.github.io/tech-doc/hab-occu.html>.
