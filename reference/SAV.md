# Chesapeake Bay Submerged Aquatic Vegetation

These data are time series Submerged Aquatic Vegetation (SAV) in the
Chesapeake Bay.

## Usage

``` r
SAV
```

## Format

Data set contains 185 rows and 4 columns

- Value: Value of variable `Var`.

- Units: Units of variable `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred. All
  data were sampled in southern Gulf of Maine.

- Time: Year of sampling.

More information including processing and indicator derivation steps are
available at
<https://noaa-edab.github.io/tech-doc/submerged-aquatic-vegetation.html>.
