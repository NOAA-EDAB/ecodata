# Bottom temperature - High Resolution

Time series of annual modeled bottom temperatures on the Northeast
Continental Shelf from 1959-pres.

## Usage

``` r
bottom_temp_model_anom
```

## Format

Data contain 1280 rows and 5 columns.

- Var: Specifies variable of interest.

- Value: Value of variable `Var`.

- Units: Units of variable `Var`.

- EPU: Ecological Production Unit (EPU) from which data were collected
  or aggregated. An EPU of `All` means that the scope of the data set is
  the entire Northeast US Continental Shelf.

- Time: Year.

## Details

Links to source data, further data definitions, and extended methodology
are available at
