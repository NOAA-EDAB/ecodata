# Warm core ring births

Data include timeseries of warm core ring births on the Northeast U.S.
Shelf.

## Usage

``` r
wcr
```

## Format

Data set contains 43 rows and 5 columns

- Time: Year.

- Var: Number of warm core rings.

- Value: Value of variable `Var`.

- Units: Unit of Variable `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

## Details

More information including processing and indicator derivation steps are
available at
<https://noaa-edab.github.io/tech-doc/warm-core-rings.html>.
