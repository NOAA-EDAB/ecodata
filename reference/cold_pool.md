# Cold pool index

The cold pool is an area of relatively cold bottom water that forms on
the US northeast shelf in the Mid-Atlantic Bight.

## Usage

``` r
cold_pool
```

## Format

A data set containing 384 rows and 4 columns.

- Time: Year

- Value: Value of variable.

- Var: Cold Pool model Temperature and volume parameters.

- EPU: Ecological Production Unit (EPU) where data originated. Here
  "MAB" refers to the Mid-Atlantic Bight.

- Source: Data source used
