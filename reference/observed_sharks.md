# NE Observer program Shark data

NE Observer program Shark data

## Usage

``` r
observed_sharks
```

## Format

These data contain 232 rows and 4 columns.

- Var: Shark category.

- Value: Value of variable `Var`.

- Time: Time step of `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

## Details

Methods used to calculate data are available at
<https://noaa-edab.github.io/tech-doc>.
