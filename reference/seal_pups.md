# Seal Pups counts in New England

Data include timeseries of seal pup count on four colonies.

## Usage

``` r
seal_pups
```

## Format

Data set contains 340 rows and 4 columns

- Time: Year.

- Var: Variable and colony name.

- Value: Value of variable `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred. All
  data were sampled in southern Gulf of Maine.

## Details

More information including processing and indicator derivation steps are
available at <https://noaa-edab.github.io/tech-doc.html>.
