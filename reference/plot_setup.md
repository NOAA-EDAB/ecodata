# Setup all plot defaults

Single function setting up plot defaults for SOE, catalog,
presentations, and other products

## Usage

``` r
plot_setup(shadedRegion = shadedRegion, report = report)
```

## Arguments

- shadedRegion:

  Numeric vector. Years denoting the shaded region of the plot (most
  recent 10), passed from plot function

- report:

  Character string. Which SOE report ("MidAtlantic", "NewEngland"),
  passed from plot function

## Value

list of settings
