# Highly Migratory Species Catch per Unit Effort

Data include catch per unit effort for all HMS species.

## Usage

``` r
hms_cpue
```

## Format

3600 rows and 4 columns.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

- Time: Year.

- Var: Species.

- Value: Value of variable `Var`.

## Details

More information available at
<https://noaa-edab.github.io/tech-doc/highly-migratory-species-landings.html>.
