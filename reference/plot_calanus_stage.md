# plot calanus_stage

Create bar plots of Calanus zooplankton stages over time

## Usage

``` r
plot_calanus_stage(shadedRegion = NULL, report = "MidAtlantic")
```

## Arguments

- shadedRegion:

  Numeric vector. Years denoting the shaded region of the plot (most
  recent 10)

- report:

  Character string. Which SOE report ("MidAtlantic", "NewEngland")

## Value

ggplot object
