# Stellwagen Bank Sandlance

These data are time series sandlance collection and co-location.

## Usage

``` r
sandlance
```

## Format

Data set contains 61 rows and 4 columns

- Value: Value of variable `Var`.

- Units: Units of variable `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred. All
  data were sampled in southern Gulf of Maine.

- Time: Year of sampling.

More information including processing and indicator derivation steps are
available at <https://noaa-edab.github.io/tech-doc/sandlance.html>.
