# Forage fish energy density

Forage fish energy density comparison between studies in the 80s and
90s.

## Usage

``` r
energy_density
```

## Format

A data set containing 264 rows and 4 columns.

- Year: Time.

- Var: Species / Season / Variable

- Value: Value of Variable

- EPU: Ecological Production Unit (NA).
