# Harbor porpoise bycatch

Data included here show the 5 year moving average for harbor porpoise
bycatch estimates and potential biological removal (PBR) levels in the
Northeast United States. These data were extracted from NEFSC Center
Reference Documents and marine mammal stock assessments.

## Usage

``` r
harborporpoise
```

## Format

160 rows and 4 columns.

- Time: Year

- Var: Bycatch variable

- Value: Value of variable

- EPU: Shelfwide indicator

## Details

Links to source data, further data definitions, and extended methodology
are available at
<https://noaa-edab.github.io/tech-doc/harbor-porpoise-bycatch.html>
