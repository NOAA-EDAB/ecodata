# Highly Migratory Species Category

Data include Species codes and names from HMS species groups.

## Usage

``` r
hms_category
```

## Format

30 rows and 4 columns.

- SP_CODE: Numeric species specific code.

- COMMON_NAME: Common species name.

- SP_CATEGORY: Grouping that the species falls into.

- COMMON_POP: Common species name from Southwest POP.
