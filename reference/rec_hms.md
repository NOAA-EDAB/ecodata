# Recreational shark fishing landings

Shark landings reported using Marine Recreational Information Program
(MRIP).

## Usage

``` r
rec_hms
```

## Format

These data contain 246 rows and 4 columns.

- Var: HMS species category.

- Value: Value of variable `Var`.

- Time: Time step of `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

## Details

Methods used to calculate data are available at
<https://noaa-edab.github.io/tech-doc/recreational-shark-fishing-indicators.html>.
