# Oyster aquaculture

These data contain time series of number of oysters, the area of oyster
lease areas and number of oysters per area for States in the Northeast.

## Usage

``` r
aquaculture
```

## Format

A data set containing 192 rows and 4 columns.

- Var: Category of oyster data, production (Pieces), Shellfish lease
  Acres (Area), Production/Area.

- Value: Value of variable `Var`.

- Region: Specific state of oyster production.

- Time: Year.
