# Forage Anomaly

Data include expected mean and uncertainty of forage anomaly from the
ecomon survey.

## Usage

``` r
forage_anomaly
```

## Format

240 rows and 4 columns.

- Value: Value of variable `Var`.

- Var: Variable calculated.

- Time: Time step of `Var`.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

## Details

Methods used to calculate data are available at
<https://noaa-edab.github.io/tech-doc/plankton-diversity.html>.
