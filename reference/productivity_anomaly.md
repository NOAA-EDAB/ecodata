# Small fish per large fish biomass anomaly

These data represent estimates of recruitment success for species
sampled in the NEFSC Bottom Trawl Survey and those calculated for stock
assessments.

## Usage

``` r
productivity_anomaly
```

## Format

Data frame with 8241 rows and 5 columns.

- Var: Common name of surveyed species. If data reflect the aggregated
  shelf-wide anomaly, then `NE LME` is included with the species name.

- EPU: EPU where sampling occurred.

- Time: Year.

- Value: Small fish per large fish biomass anomaly.

- Units: Units of variable `Var`.

## Details

Detailed information regarding the derivation of this indicator are
present at
<https://noaa-edab.github.io/tech-doc/fish-productivity-indicator.html>.
