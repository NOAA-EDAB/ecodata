# Forage Index

Data include forage fish biomass index from predator diet data from
NEFSC bottom trawl survey.

## Usage

``` r
forage_index
```

## Format

888 rows and 5 columns.

- Value: Value of variable `Var`.

- Var: Variable calculated.

- Time: Year.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

- Units: grams per stomach.

## Details

Methods used to calculate data are available at
<https://noaa-edab.github.io/tech-doc/forage_index.html>.
