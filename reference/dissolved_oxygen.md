# Dissolved Oxygen from FishBOT

Gridded observations of dissolved oxygen from FishBOT

## Usage

``` r
dissolved_oxygen
```

## Format

A data set containing 14736 rows and 5 columns.

- Time: Date

- Lon: Longitude

- Lat: Latitude

- Value: Value of variable.

- Var: Dissolved Oxygen obervations: (measured = 'dissolved_oxygen',
  count = 'dissolved_oxygen_count', minimum = 'dissolved_oxygen_min', or
  maximum = 'dissolved_oxygen_max')
