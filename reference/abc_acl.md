# ABC or ACL for Managed Stocks

These data contain times series of ABC or ACL and Catch for Mid-Atlantic
and New England managed stocks

## Usage

``` r
abc_acl
```

## Format

A data set containing 1930 rows and 5 columns.

- Time: Management year.

- Value: Value of variable `Var`

- Var: Quota or Catch for each Fisheries Management Plan.

- EPU: Ecological Production Unit (EPU) where species is managed. Here
  "MAB" refers to the Mid-Atlantic Bight and "NE" refers to New England.

- Units: Metric tons (mt)
