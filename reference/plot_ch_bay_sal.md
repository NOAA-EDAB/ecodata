# plot ch_bay_sal

Plot Chesapeake Bay salinity daily time series.

## Usage

``` r
plot_ch_bay_sal(shadedRegion = NULL, report = "MidAtlantic")
```

## Arguments

- shadedRegion:

  Numeric vector. Years denoting the shaded region of the plot (most
  recent 10)

- report:

  Character string. Which SOE report ("MidAtlantic" only, default)

## Value

ggplot object
