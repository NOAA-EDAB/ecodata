# Benthos Index

Data include Megabenthos and Macrobenthos biomass and center of gravity
indices from predator diet data from NEFSC bottom trawl survey.

## Usage

``` r
benthos_index
```

## Format

n rows and 5 columns.

- Value: Value of variable `Var`.

- Var: Variable calculated.

- Time: Year.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

- Units: grams per stomach.

## Details

Methods used to calculate data are available at
<https://noaa-edab.github.io/tech-doc/benthos_index.html>.
