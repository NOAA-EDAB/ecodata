# Sea surface temperature seasonal transiition dates

Data include transition day of year.

## Usage

``` r
trans_dates
```

## Format

1599 rows and 4 columns.

- EPU: Ecological Production Unit (EPU) where sampling occurred.

- Time: Year.

- Var: Variable describing shannon diversity.

- Value: Value of variable `Var`.
